# LinxFrontend

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.17.

Project - FrontEnd Angular

## Installation and run - Development server

1. Clone this repository: `https://github.com/Deyveson/frontend-linx.git`
2. `cd` into the `frontend-linx` folder and run the following commands:
    1. Install `npm install`
    2. Run `npm start` for a dev server
3. Navigate to `http://localhost:9001/search?terms=teste`. The app will automatically reload if you change any of the source files.
